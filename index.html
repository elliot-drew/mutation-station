Mutation Station

Produce mutation inputs for a variety of computational mutagenesis tools from a protein sequence.

  1. Select a residue, a residue range, residue types to mutate or residue types to mutate to.
  2. Pick the tool/website you want to generate input for.
  3. Click Generate!
Copy the text produced, or click Download to get a text file containing the generated input.
